| Модель                  | Зачем нужна                                                 | Важные поля                                             | Сценарии из ТЗ                                                     |
| ----------------------- | ----------------------------------------------------------- | ------------------------------------------------------- | ------------------------------------------------------------------ |
| `Warehouse`             | Склад с адресом и контактами; умеет считать свободные боксы | `title, city, address, phone, work_hours`               | “куда везти вещи”, показать адреса; “сколько боксов осталось”      |
| `Box`                   | Бокс с размерами и ценой                                    | `length_m, width_m, height_m` + `volume_m3`             | Расчёт цены: `м³ * 750`, выбор бокса                               |
| `StorageRule`           | Правила хранения (разрешено/запрещено)                      | `rule_type, title`                                      | “можно ли жидкости” + список allowed/forbidden                     |
| `PrivacyPolicyDocument` | PDF для согласия на ПД                                      | `file, is_active`                                       | “не получить штраф” — согласие при оформлении                      |
| `PromoCode`             | Единственная “фича”                                         | `code, discount_percent, starts_at, ends_at`            | сезонные промокоды                                                 |
| `AdCampaign`            | Трекинг рекламы по коду                                     | `code`                                                  | “сколько людей сделали заказы с рекламы”                           |
| `UserProfile`           | ЛК как на макете (телефон/аватар)                           | `user, phone, avatar`                                   | личный кабинет пользователя                                        |
| `Rental`                | Аренда бокса (по умолчанию 1 месяц) + QR                    | `user, box, start_date, end_date, status, access_token` | заказ бокса, QR для открытия, просрочка/потеря через 6 мес         |
| `DeliveryTask`          | “куда ехать” для доставщика                                 | `rental, from/to_address, planned_date`                 | выполнить вывоз/доставку, видеть телефон                           |
| `EmailNotification`     | Лог напоминаний/писем                                       | `kind, is_sent, sent_at`                                | письма за 30/14/7/3 дня + ежемесячно после просрочки + “что будет” |
